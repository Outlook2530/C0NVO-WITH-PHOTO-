How to Get Facebook Conversation ID
  1. Go to Facebook Messenger
  2. Open the conversation you want to target
  3. Look at the URL: https://www.facebook.com/messages/t/1000123456789
  4. The number after /t/ is the Conversation ID
Note: Make sure you have permission to message the recipient.
